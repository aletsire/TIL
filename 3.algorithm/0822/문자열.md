## 1. 패턴 매칭

### 1.1 고지식한 패턴 검색 알고리즘

```python
p = 'is'
t = 'This is nothing'
M = len(p)
N = len(t)

def solution(p, t):
    i = 0
    j = 0
    while j < M and i < N:
        if t[i] != p[j]:
            i = i-j
            j = -1
            i += 1
            j += 1
        if j ==M:
            return i
        else:
            return -1
```



### 1.2 KMP 알고리즘

- 불일치가 발생한 텍스트 스트링의 앞 부분에 어떤 문자가 있는지를 미리 알고 있으므로, 불일치가 발생한 앞부분에 대하여 다시 비교하지 않고 매칭을 수행



### 1.3 보이어-무어 알고리즘

- 패턴의 오른쪽 끝에 있는 물자가 불일치하고 이문자가 패턴 내에 존재하지 않는 경우, 패턴의 길이만큼 이동
- 패턴 내에 존재하는경우 문자를 찾아서 그 거리만큼 이동
